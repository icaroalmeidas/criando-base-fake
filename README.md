## Dimension Dataset Datalake
Neste projeto, manipulei um arquivo em parquet com mais de um milhão de linhas, separando-o em vários DataFrames, simulando um datalake.

### DataFrames Criados
- `df_people`
- `df_cities`
- `df_countries`
- `df_addresses`
- `df_jobs`
- `df_age`
- `df_contacts`
- `df_temporal`
- `df_registers`

**Observação:** Todos os DataFrames foram salvos como parquet.
